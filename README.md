# Bitcoin Market Sentiment Analysis

## Overview
This project analyzes the relationship between Bitcoin market sentiment and Hyperliquid trader performance using historical trading data and the Bitcoin Fear & Greed Index.

## Objectives
- Analyze trader profitability under different market sentiments.
- Identify hidden trading patterns.
- Perform statistical hypothesis testing.
- Build a Decision Tree model to predict trade outcomes.

## Dataset
- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trading Data

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Methods
- Data Cleaning
- Exploratory Data Analysis
- Correlation Analysis
- Hypothesis Testing (t-test)
- Decision Tree Classification
- Hyperparameter Tuning using GridSearchCV

## Results
- Market sentiment alone is not a statistically significant predictor of profitability.
- Trade size strongly influences transaction fees.
- Decision Tree achieved approximately 84.8% test accuracy in predicting trade outcomes.
